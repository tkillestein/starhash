# `starhash` data

This folder contains the data files referenced in the `starhash` paper, along with
their original source and provenance. Both catalogs are based on versions downloaded
30th March 2026, and are unlikely to be updated.

Files are distributed as `gzip`-compressed CSV files, and can be manipulated with your
favourite array/dataframe library.

If you make use of these files, please
ensure you include the relevant citations and/or acknowledgements for these crucial 
community resources.

### `tns_public_objects.starhashed.csv.gz`

Retrieved from the [Transient Name Server](https://wis-tns.org), via the daily staging
links.

### `nasa_exoplanet_archive.csv.gz`

Retrieved from the [NASA Exoplanet Archive](https://exoplanetarchive.ipac.caltech.edu/),
based on the Planetary Systems table.
